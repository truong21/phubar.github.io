Projects

AirBnB Clone

Search Page With Filtering

This project recreates a search page that imitates the filtering and database management of a AirBnB. Using pair programming and test driven development, the project is completed in four stages. Stage 1 consists of creating a custom console in Python to help test functionality. Stage 2 sees the implementation of storage system in both a JSON text file storage system and ORM database storage system. Stage 3 and 4 utilizes Jinja2 and JQuery to dynamically populate and filter places for rent based on search parameters fed into a REST API.
